Project Summary
The Empresa San Luis project is focused on developing a corporate website for a prominent energy company specializing in hydrocarbons, lubricants, supplies, and transportation. The website will reflect the company’s brand identity as outlined in their visual identity document and will be built as a Progressive Web App (PWA) to improve user experience and accessibility.

Project Module Description
The project includes the following key modules:

Header: Navigation with company logo, links, and a contact button.
Hero Section: A dynamic carousel showcasing the company’s mission and services.
Business Units: Interactive cards representing the four business divisions.
About Us: Information on the company’s history and values.
Sustainability: Initiatives promoting environmental responsibility.
Contact: A functional form for user inquiries.
Footer: Institutional links and company information.

File Description Inventory
index.html: The main structure of the website, containing all sections.
logo.png: The official logo of San Luis, displayed in the header.
manifest.json: Contains metadata for the PWA, such as name, icons, and theme colors.
package.json: Lists project dependencies and scripts for building and linting.
script.js: Contains JavaScript functionality for interactive elements.
style.css: Styles for the website, implementing Material You design principles and brand identity.
sw.js: Service Worker script for caching assets and offline support.
template_config.json: Holds configuration settings for templates used in the project.
Technology Stack
HTML5: For semantic structure.
CSS3: For styling, using modern techniques and variables.
JavaScript: Vanilla JS for interactivity.
Progressive Web App (PWA): For enhanced performance and offline capabilities.
Usage
Install Dependencies: Run pnpm install in the app/frontend directory to install required packages.
Build the Project: Execute pnpm run build to compile the project.
Run Lint Check: Use pnpm run lint to ensure code quality.
Deploy: Follow deployment instructions to publish the website.
